# assets

Place the README banner image here as **`banner.png`**.

The README references it as `assets/banner.png`:

```markdown
<img src="assets/banner.png" alt="LinkedOut — the LinkedIn feed, minus the slop" width="900">
```

Recommended: PNG, ~1600–2000 px wide. Once `banner.png` is committed here, it
renders automatically at the top of the README.
